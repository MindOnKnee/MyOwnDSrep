Цели: провести разведывательный анализ данных по базе ресторанов, провести анализ признаков и сформировать дополнительные для обучения модели.

Данные: 10 колонок, 40000 строк, пустые значения в 3 колонках.

Этапы работы: первичная обработка данных,  
заполнение пустых значений, оценка уникальных значений номинативных переменных, 
корреляционный анализ, анализ номинативных переменных, feature engineering, выводы, обучение модели.
